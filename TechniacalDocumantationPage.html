HTML Basics

HTML stands for HyperText Markup Language.

It is the standard language for creating web pages.

HTML describes the structure of a web page.

HTML elements are represented by tags.

Common HTML tags include &alt;div>, &alt;p>, and <a>.

<div>This is a div element</div>
  • HTML is the foundation of web development.
  • It is easy to learn and use.
  • HTML elements form the building blocks of web pages.
CSS Basics

CSS stands for Cascading Style Sheets.

It describes how HTML elements are to be displayed.

CSS saves a lot of work by controlling the layout of multiple web pages.

External stylesheets are stored in CSS files.

CSS can be added to HTML elements in 3 ways: Inline, Internal, and External.

body {
  background-color: lightblue;
}
  • CSS makes web pages look attractive.
  • It allows for flexible designs.
  • CSS is easy to learn and implement.
JavaScript Basics

JavaScript is programming language for the web.

It makes web pages interactive

JavaScript can update and change both HTML and CSS.

It can calculate, manipulate and validate data.

JavaScript is easy to learn and implement.

function greet() {
            alert('Hello, world');
          }
  • JavaScript is an essential web technology.
  • It enhances user experience.
  • JavaScript works on both client and server sides.
